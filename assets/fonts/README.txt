UniversNextforHSBC Font Files

To use the UniversNextforHSBC font properly, please place the following font files in this directory:

1. UniversNextforHSBC-Regular.ttf
2. UniversNextforHSBC-Bold.ttf
3. UniversNextforHSBC-Light.ttf

These font files are proprietary to HSBC and not included in this repository.

If you don't have access to these files, the app will fallback to a similar system font automatically.

For developers: The font configuration is set in pubspec.yaml and theme.dart files.
